# Лабораторная работа №5
## Работа с DOM-деревом и событиями в JavaScript.


## Цель работы
### Ознакомиться с основами взаимодействия JavaScript с DOM-деревом на основе веб-приложения для учета личных финансов.

### Необходимо было создать структуру файлов:
 `index.html`;

 `style.css`;

 `src/index.js`;

 `src/transactions.js`;

 `src/ui.js`;

 `src/utils.js`.

### в приложении необходимо было добавить транзакции, возможность удолять транзакции, просмотривать полное описание транзакций, а так же видеть полную сумму транзакций.
