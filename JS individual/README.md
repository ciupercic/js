## Особенности

 ** Основы арифметики**: +, −, ×, ÷  
 ** Поддержка десятичной системы счисления **: Числа с плавающей запятой  
 ** Удалить (⌫) **: Удалить последнюю цифру  
 **Очистить (C)**: Сброс настроек калькулятора  
 ** Поддержка клавиатуры **: Ввод чисел и операторов  
 ** Обработка ошибок **: Показывает "Ошибку" для недопустимых выражений  
 ** Адаптивный **: Работает на настольных компьютерах, планшетах, мобильных устройствах

### Сочетания клавиш
- `0-9, .` → Ввод чисел
- `+, -, *, /` → Операторы
- `Enter` или `=` → Вычислить
- `Backspace` → Удалить
- `Escape` → Очистить

## Примеры использования

### Пример 1: Простое сложение
```
Вход: 5 + 3 =

## Технические подробности

### Архитектура: Шаблон модуля (IIFE)
``
калькулятор констант javascript = (() => {
    // ЧАСТНЫЕ переменные и методы
    пусть выражение = ";
    const Добавить номер = (num) => { /* ... */ };
    
    // ОТКРЫТЫЙ интерфейс
    возвращает {
        init: () => { /* Инициализировать */ }
    };
})();
```

### Основные методы
- `Добавить номер(num)` - Добавить цифру или десятичное число
- `Добавить оператор(op)` - Добавить +, −, ×, ÷
- `Удалить символ()` - Удалить последний символ
- `очистить()` - Сбросить настройки калькулятора
- `рассчитать()` - Вычислить выражение

**Версия:** 1.0 
**Основные итоги работы:**
1.  **Закрепление практических навыков:** На практике освоена работа с чистым языком JavaScript (Vanilla JS) без применения сторонних фреймворков. Это позволило глубже понять внутренние механизмы взаимодействия скриптов с веб-страницей.
2.  **Объектная модель документа (DOM):** Реализован динамический рендеринг элементов интерфейса и манипуляция ими «на лету» в ответ на действия пользователя.
3.  **Оптимизация архитектуры:** Приложение структурировано с использованием модульного подхода, что обеспечивает читаемость, расширяемость и легкость поддержки кода. Использование паттернов проектирования (IIFE/Module) помогло инкапсулировать внутреннюю логику вычислений и защитить глобальную область видимости.
4.  **Управление событиями и валидация:** Написаны отказоустойчивые алгоритмы для обработки пользовательского ввода (включая защиту от некорректных математических выражений и двойных точек в дробных числах) и реализован эффективный механизм делегирования событий.
