# Лабораторная работа №1
### Инструкция по запуску проекта.
1. Создать папку проекта.  
2. Созить файл index.html.
3. Создать файл script.js.
4. Вставить соответствующий код в каждый файл.
5. Открыть index.html в браузере
или запустить через Live Server.
### Описание лабораторной работы - изучить.
1. структуру HTML-документа;
2. подключение внешнего JavaScript-файла;
3. работу с переменными;
4. типы данных в JavaScript;
5. условные операторы (if / else);
6. цикл for;
7. вывод информации в консоль браузера.

### Краткая документация к проекту.
В ходе работы была создана HTML-страница с подключённым JavaScript-кодом, который выполняет ввод данных пользователя, проверку условий и повторяющиеся операции.

`в корневой папке project расположены два файла — index.html и script.js.
Файл index.html содержит HTML-разметку страницы, а script.js реализует программную логику на языке JavaScript.`

`index.html`
* Подключает внешний файл script
* jsСодержит базовую HTML-структуру
* Выполняет встроенный JavaScript-код

`script.js`
* вывод сообщений через alert() и console.log();
* объявление переменных (let);
* ввод данных через prompt();
* условную конструкцию if / else;
* цикл for.


### Код проекта.
`index.html`

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <script src="script.js"></script>
    <title>Привет, мир!</title>
</head>
<body>
<script>
    alert("Привет, мир!");
    console.log("Hello, console!");
</script>
</body>
</html>
```
`script.js`
```javascript
alert("Этот код выполнен из внешнего файла!");
console.log("Сообщение в консоли");
let name = "Ольга";
let birthYear = "2002";
let isStudent = true;
console.log("Имя:", name);
console.log("Год рождения:", birthYear);
console.log("Студент:", isStudent);
let score = prompt("Введите ваш балл:");
if (score >= 90) {
 console.log("Отлично!");
} else if (score >= 70) {
 console.log("Хорошо");
} else {
 console.log("Можно лучше!");
}

for (let i = 1; i <= 5; i++) {
 console.log(`Итерация: ${i}`);
}
```
### Пример работы программы:
Если пользователь вводит `95`, то результат в консоле будет выводит `"Олично!"`.
Если пользователь будет вводить `80`, то результат в консоле будет выводить `"Хорошо"`.
Если пользователь будет вводить `50`, то результат в консоле будет выводить `"Можно лучше!"`.

### Ответы навопросы.
`var` — устаревший способ объявления переменной. Сейчас не так востребована так, как может вызывать ошибки и работает менее безопасно.

`let` - это современный вариант которе может менять значение.
Нельзя объявить два раза одну и ту же переменную. Работает только внутри блока `{ }`

`const`- Нельзя менять значение после создания Используется для постоянных значений.
#### Что такое неявное преобразование типов?
Это когда `JavaScript` сам меняет тип данных автоматически, например если ввести 2+3 то в результате он выведит 23.
#### Разница между `==` и `===`
`==`— (обычное сравнение) сравнивает только значение

`===` — (строгое сравнение)сравнивает значение и тип

`5 == "5"`   // true

`5 === "5"`  // false
